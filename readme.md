# Task 2
### № 2.1

Описание переменных:
    `epsilon` - заданная точность вычисления корня
    `x` - число, из которого вычисляется квадратный корень
    `current` - текущее приближение корня
    `next` - следующее приближение корня

Описание функций:
    `absolute(double value)` - возвращает модуль числа

Основной алгоритм в main() реализует итерационную формулу Ньютона

Описание действия программы:
  Программа считывает точность epsilon, затем последовательность чисел x. Для каждого x вычисляется квадратный корень методом Ньютона до достижения заданной точности. Результаты выводятся с форматированием %.10g.

### № 2.3

Описание переменных:
    `prev2`, `prev1`, `current` - переменные для итеративного вычисления
    `i` - индекс числа Фибоначчи

Описание функций:
    `fibonacci_iterative(int n)` - итеративная версия вычисления чисел Фибоначчи
    `fibonacci_recursive(int n)` - рекурсивная версия вычисления чисел Фибоначчи

Описание действия программы:
  Программа считывает последовательность индексов i и для каждого вычисляет i-е число Фибоначчи двумя способами. Позволяет сравнить производительность итеративного и рекурсивного подходов.

### № 2.5

Структуры данных:
    `WordNode` - узел списка слов
        ◦ `word` - указатель на строку с словом
        ◦ `next` - указатель на следующий узел
Описание функций:
    `create_node(const char* word)` - создает новый узел списка
    `add_to_list(WordNode** head, const char* word)` - добавляет слово в конец списка
    `remove_matching_words(WordNode** head)` - удаляет слова, совпадающие с последним
    `print_list(WordNode* head)` - выводит список слов
Описание действия программы:
  Программа считывает строку, разбивает ее на слова, сохраняет в связный список, затем удаляет все слова, совпадающие с последним словом (кроме самого последнего), и выводит результат.

### № 2.6

Структуры данных:
    `TreeNode` - узел бинарного дерева поиска
        ◦ `key` - целочисленный ключ
        ◦ `left, right` - указатели на левое и правое поддеревья
Описание функций:
    `create_node(int key)` - создает новый узел дерева
    `insert(TreeNode* root, int key)` - вставляет ключ в дерево
    `delete(TreeNode* root, int key)` - удаляет ключ из дерева
    `search(TreeNode* root, int key)` - ищет ключ в дереве
    `find_min(TreeNode* root)` - находит минимальный элемент
    `glubina(TreeNode* root)` - вычисляет глубину дерева
Описание действия программы:
  Программа реализует интерактивную работу с бинарным деревом поиска. Поддерживает операции:
    +число - добавление элемента
    -число - удаление элемента
    ?число - проверка наличия элемента
  В конце выводит глубину дерева.

### № 2.7

Описание переменных
    `curlex` - текущая лексема (символ)
    `result` - результат вычисления выражения
Описание функций:

    `getlex(void)` - выделяет следующую лексему из входного потока
    
    `expr(void)` - разбор выражения (сложение/вычитание)
    
    `term(void)` - разбор слагаемого (умножение/деление)
    
    `factor(void)` - разбор множителя (степень)
    
    `power(void)` - разбор степени (числа/скобки)
    
    `error(void)` - обработка ошибок
    
Описание действия программы:
Программа-калькулятор, вычисляющая арифметические выражения методом рекурсивного спуска. Поддерживает операции: +, -, *, /, ' (степень), скобки. Включает проверку ошибок (деление на ноль, отрицательная степень) и интерактивный режим работы.
